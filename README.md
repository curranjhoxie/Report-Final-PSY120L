
This repository contains the published HTML version of my final reproducible report for PSY 120L.
This project explored how the nature of the feedback given by a professor influences students’ perceptions of their own self-efficacy and motivation to succeed academically. It also explored whether students’ responses to this feedback are influenced by parenting style.
This report was produced using Quarto and R.
The published report file is index.html.
The data set is not included in this repository.
